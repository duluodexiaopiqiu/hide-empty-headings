# Hide Empty Headings

Automatically hide empty headings in Obsidian's reading mode.

## Features

- Automatically hides headings with no content below them in reading mode
- Triggers automatically when switching notes

## Installation

Copy the entire folder (this code project) to `.obsidian\plugins`

![Installation Guide](aseets/pic1.png)

## Usage

No configuration required after installation. The plugin works automatically when you open any note in reading mode.
